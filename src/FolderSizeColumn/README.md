# Folder Size Column

This project is a shell extension that creates a "Folder Size" column in Windows Explorer. It reads the data generated by the `SizeCacheService` to display the sizes.

## Prerequisites

1.  **.NET 8 SDK**: You must have the .NET 8 SDK installed.
2.  **SizeCacheService**: The `SizeCacheService` project must be built, as this extension reads the `sizecache.json` file it produces.

## 1. Build the DLL

First, compile the project to create the necessary DLLs.

1.  **Open an Administrator Command Prompt or PowerShell.**
2.  **Navigate to this project's directory:**
    ```sh
    cd c:\path	o\your\windows-customizer\src\FolderSizeColumn
    ```
3.  **Run the build command:**
    ```sh
    dotnet build --configuration Release
    ```
This will create the `FolderSizeColumn.dll` and `FolderSizeColumn.comhost.dll` in the `bin\Release` directory.

## 2. Register the Shell Extension

This is the most important step. You are registering a COM server that will be loaded by `explorer.exe`. This **must** be done from an Administrator terminal.

1.  **Open an Administrator Command Prompt or PowerShell.**
2.  **Navigate to the output directory:**
    ```sh
    cd bin\Release
    ```
3.  **Run the registration command:**
    ```sh
    regsvr32.exe FolderSizeColumn.comhost.dll
    ```
You should see a message box confirming that the registration succeeded.

## 3. Restart Explorer and Use the Column

For the changes to take effect, you must restart Windows Explorer.
1.  Open Task Manager (`Ctrl+Shift+Esc`).
2.  Find "Windows Explorer" in the "Processes" tab.
3.  Right-click it and select "Restart".

Now, open a folder that is being monitored by the `SizeCacheService`:
1.  Right-click on any column header (e.g., "Name", "Date modified").
2.  Click "More...".
3.  Scroll down the list and find "**Folder Size**".
4.  Check the box and click OK.

The new column will appear, displaying the cached sizes for the folders.

## Unregistering the Shell Extension

If you want to remove the shell extension, you must un-register it.

1.  **Open an Administrator Command Prompt or PowerShell.**
2.  **Navigate to the `bin\Release` directory.**
3.  **Run the un-registration command:**
    ```sh
    regsvr32.exe /u FolderSizeColumn.comhost.dll
    ```
4.  Restart Windows Explorer again.
