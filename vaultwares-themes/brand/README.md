# Brand

This folder contains the maintainable VaultWares brand source.

| File / folder | Purpose |
| --- | --- |
| `brand-guide.md` | Human-readable and agent-readable brand guide. |
| `philosophy.md` | Long-form brand philosophy and positioning. |
| `tokens/` | Canonical design tokens and platform mappings. |
| `i18n/` | Bilingual brand strings and copy patterns. |
| `legacy-notes/` | Preserved older notes that have useful provenance but are no longer canonical. |

Generated brand-guide exports should not live here. If a visual guide is needed,
update `brand-guide.md` and the optional demo in `examples/brand-guide/`.
